VitaSenseAI: Health Risk Prediction Web App

Project Overview
VitaSenseAI is a web-based application that predicts an individual's health risk based on lifestyle, demographic, and clinical features. It combines Machine Learning (Random Forest Classifier) and Knowledge Representation & Reasoning (KRR) rules to provide interpretable predictions. Users can monitor their health remotely and get early warnings without visiting a hospital.

Features:
Predicts health risk: Low / Medium / High
Combines ML predictions with rule-based reasoning for better interpretability
Provides explanations for predicted risks
User-friendly, trendy web interface with responsive design

Allows input of:
    Age, Weight, Height
    Exercise frequency
    Sleep hours
    intake
    Smoking and alcohol habits
    Marital status
    Profession

Technologies Used
    Frontend: HTML, CSS, JavaScript
    Backend: Python, Flask
    Machine Learning: scikit-learn (Random Forest)
    Data Processing: pandas, numpy
    Model Serialization: joblib
